# Stock Data Analysis Project

This project was completed as part of my Data Analysis Internship.

## Overview
The project focuses on analyzing stock price data to uncover trends, patterns, and insights using data analysis and machine learning techniques.

## Key Tasks Completed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Regression Analysis
- Clustering (K-Means)
- Classification Models
- Power BI Dashboard

## Machine Learning Models
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest

## Dashboard
An interactive Power BI dashboard was created to visualize:
- Stock price trends
- Trading volume
- Price relationships
- Market behavior

## Tools Used
- Python (Pandas, Matplotlib, Scikit-learn)
- Power BI

## Key Insights
- Strong correlation between stock price variables
- Predictive models achieved high accuracy
- Clear patterns in stock price movements


This project reflects my journey in learning and applying real-world data analysis skills.
